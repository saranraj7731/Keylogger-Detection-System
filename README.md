# Keylogger Detection & System Monitoring

## Objective

To monitor running processes and detect suspicious applications that may behave like keyloggers.

## Tools Used

* Python
* Tkinter
* Psutil

## Features

* Live Process Monitoring
* CPU Usage Monitoring
* RAM Usage Monitoring
* Suspicious Process Detection
* Alert Logging
* Log Viewer

## Detection Logic

The application checks running processes against predefined suspicious keywords such as:

* keylogger
* logger
* spy
* hook
* capture

## Conclusion

This project demonstrates defensive cybersecurity techniques by monitoring system activity and detecting potentially suspicious processes.
